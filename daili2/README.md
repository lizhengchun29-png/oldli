# Socks5代理管理器

这是一个用Python开发的Windows系统代理管理工具，可以爬取、验证和管理Socks5代理。

## 功能特点

- 从多个代理源爬取Socks5代理
- 验证代理的有效性
- 将有效代理保存到SQLite数据库
- 设置系统全局代理
- 可视化界面，操作简便

## 安装依赖

```bash
pip install -r requirements.txt
```

## 使用方法

1. 运行程序：

```bash
python proxy_manager.py
```

2. 主要功能：

- **爬取代理**：从下拉菜单选择代理源，点击"爬取代理"按钮
- **验证列表中IP**：验证当前列表中的所有代理
- **验证数据库中IP**：从数据库加载代理并验证
- **提取数据库中IP**：将数据库中的代理加载到列表中
- **清空列表**：清空当前代理列表
- **设置为全局代理**：右键点击列表中的代理，选择"设置为全局代理"
- **取消代理设置**：右键点击列表，选择"取消代理设置"

## 代理源

程序支持从以下代理源爬取Socks5代理：

- free-proxy-list
- spys-one
- socks-proxy
- geonode
- proxyscrape

## 系统要求

- Windows操作系统
- Python 3.6+
- 安装所有依赖库

## 注意事项

- 代理验证过程可能需要一些时间，请耐心等待
- 设置系统代理需要管理员权限
- 代理有效性会随时间变化，建议定期验证
